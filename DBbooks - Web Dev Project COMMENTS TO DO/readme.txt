Deployment Instructions:

Create an ENV file (notepad, save as, all files, call it '.env') including the following

MONGODB_URI = mongodb+srv://benji-123:benji-123@cluster0.czli7qj.mongodb.net/?retryWrites=true&w=majority


Once done navigate to folder in CMD using cd

'npm install'

'npm start'

Open your web browser and go to localhost:3000 and the website will work from there.
